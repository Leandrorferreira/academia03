Versionamento GIT

Comandos Terminal

pwd - endereço onde estou (pasta local)
ls - mostra o que tenho na pasta
cd - mudar de pasta
mkdir - cria nova pasta
touch - cria um arquivo
clear - limpa tela
ls -la - mostra o conteudo e detalhes
---------------------------------------
Comandos Git

git init - inicializa o repositorio git
git status - mostra o estágio 
git add - adiciona no repositorio
git add . - adiciona todos os arquivos a serem commitados
git commit - confirmar com mensagem
git log - mostra um log de informações do commit 
git config user.email "seu_email" - adiciona assinatura de email
git config user.name "nome" - adiciona assinatura de nome
git push "endereço remoto".git master - empurrando o codigo para um local
git clone "endereço" - faz o download do repositorio inteiro
git diff - exibe as diferenças de modificação do arquivo

untracked - não esta adicionado no repositorio
unstaged - pronto para ser versionado 
staged - foram versionados
